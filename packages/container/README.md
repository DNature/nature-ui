# Container `@nature-ui/container`

## Installation

```sh
npm i @chakra-ui/container

# or

yarn add @chakra-ui/container
```

## Usage

```js
import Alert from '@nature-ui/container';

() => {
  return (
    <Container className='bg-gray-300' size='xl'>
      I love Nature
    </Container>
  );
};
```
